## 📋 プロジェクト概要

このプロジェクトは、日本のスタートアップ企業向けに一から構築された完全なエンタープライズウェブサイトソリューションを紹介しています。アプリケーションは、包括的なコンテンツ管理機能と多言語サポートを備えたモダンでレスポンシブなデザインが特徴です。

### 🎯 主要成果
- ✅ **フルスタック開発** デザインからデプロイメントまで
- ✅ **多言語サポート** （日本語、英語、中国語）
- ✅ **カスタムCMS統合** 非技術者向けコンテンツ管理
- ✅ **自社ホスティングインフラ** 自動デプロイパイプライン付き
- ✅ **エンタープライズグレードセキュリティ** SSL証明書とメール統合
- ✅ **100%レスポンシブデザイン** 全デバイス最適化

## 🛠️ 技術的実装

### フロントエンドアーキテクチャ
| コンポーネント | 技術 | 目的 |
|-----------|------------|---------|
| **UIフレームワーク** | React 18 + TypeScript | タイプセーフティ付きコンポーネントベースアーキテクチャ |
| **スタイリング** | Tailwind CSS | 迅速な開発のためのユーティリティファーストCSSフレームワーク |
| **国際化** | React i18next | 多言語コンテンツ管理 |
| **状態管理** | React Hooks + Context API | コンポーネント間の効率的な状態処理 |
| **ルーティング** | React Router v6 | クライアントサイドナビゲーションとルート保護 |

### バックエンド・インフラストラクチャ
| サービス | 技術 | 構成 |
|---------|------------|---------------|
| **コンテンツ管理** | Netlify CMS | ビジュアルエディタ付きGitベースヘッドレスCMS |
| **Webサーバー** | Nginx | パフォーマンス最適化付きリバースプロキシ |
| **SSL証明書** | Let's Encrypt | 自動証明書更新 |
| **メールサービス** | Zoho Mail | カスタムドメイン付きプロフェッショナルメール |
| **DNS管理** | カスタム構成 | ドメインルーティングとサブドメインセットアップ |
| **CI/CDパイプライン** | GitHub Actions | 自動テストとデプロイメント |

### 開発ツール
- **バージョン管理:** 機能ブランチワークフローでのGit
- **コード品質:** 一貫したフォーマットのためのESLint + Prettier
- **テスト:** Jest + React Testing Library
- **パフォーマンス:** Lighthouse最適化（95+スコア）
- **アクセシビリティ:** WCAG 2.1 AA準拠

## 🔧 ローカル開発セットアップ

### 前提条件
```bash
# 必要なソフトウェア
Node.js 18+
npm 8+
Git 2.30+
```

### インストール
```bash
# リポジトリのクローン
git clone [repository-url]
cd iwabayashi-website

# 依存関係のインストール
npm install

# 開発サーバー起動
npm start

# テスト実行
npm test

# 本番用ビルド
npm run build
```

### 環境設定
```bash
# .envファイル作成
REACT_APP_CMS_API_URL=your_cms_url
REACT_APP_ANALYTICS_ID=your_analytics_id
REACT_APP_EMAIL_SERVICE=your_email_service
```

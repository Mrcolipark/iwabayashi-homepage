## 🛠️ Technical Architecture

### Frontend Stack
| Technology | Implementation | Purpose |
|------------|----------------|---------|
| **React 18** | Component-based architecture | Dynamic UI with type safety |
| **TypeScript** | Full type coverage | Development error prevention |
| **Tailwind CSS** | Utility-first styling | Rapid responsive development |
| **React i18next** | Internationalization | Seamless language switching |
| **React Router v6** | Client-side routing | Multi-language URL structure |

### Backend & Infrastructure
| Service | Technology | Configuration |
|---------|------------|---------------|
| **Content Management** | Netlify CMS | Git-based headless CMS with visual editor |
| **Web Server** | Nginx | Performance-optimized reverse proxy |
| **SSL Security** | Let's Encrypt | Automated certificate renewal |
| **Email Integration** | Zoho Mail | Professional email with custom domain |
| **Domain Management** | Custom DNS | Full domain and subdomain setup |
| **CI/CD Pipeline** | GitHub Actions | Automated testing and deployment |

### Development Workflow
- **Version Control:** Git with feature branch workflow
- **Code Quality:** ESLint + Prettier for consistent formatting
- **Testing:** Jest + React Testing Library for component testing
- **Performance:** Lighthouse optimization achieving 95+ scores
- **Accessibility:** WCAG 2.1 AA compliance implementation

## 🚀 Local Development Setup

### Prerequisites
```bash
Node.js 18+
npm 8+
Git 2.30+
```

### Installation & Running
```bash
# Clone and setup
git clone [repository-url]
cd iwabayashi-website
npm install

# Development
npm start          # Start development server
npm test           # Run test suite
npm run build      # Production build
npm run lint       # Code quality check
```

### Environment Configuration
```bash
# .env file setup
REACT_APP_CMS_API_URL=your_cms_url
REACT_APP_ANALYTICS_ID=your_analytics_id
REACT_APP_EMAIL_SERVICE=your_email_service
REACT_APP_DOMAIN=iwabayashi.com
```
